# Mini-Project (Major Health Indicators-across India)

Project Overview: 
This project provides an analysis of major health indicators across India using data analytics techniques. It focuses on identifying trends, patterns, and regional disparities in key health areas such as disease prevalence, mortality rates, and healthcare outcomes. The analysis helps in understanding the overall health scenario and highlights areas that require attention and improvement.

Data source:
The dataset used in this project was sourced from Data.gov.in, which provides reliable and publicly available government data. The raw data was processed and cleaned using Microsoft Excel to ensure accuracy, consistency, and usability for analysis.

Dashboard Development:
The dashboards were developed using Microsoft Power BI, enabling the creation of interactive and visually appealing reports. Various charts, maps, and tables were used to present insights effectively, allowing users to explore trends, compare regions, and gain meaningful insights into the health indicators across India.
